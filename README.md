## Данные

Ссылка на датасет: [kaggle.com/datasets/jessicali9530/kuc-hackathon-winter-2018](https://www.kaggle.com/datasets/jessicali9530/kuc-hackathon-winter-2018)

## Основная задача

Прогнозировать значение **`rating`** на основе текста **`review`**. Это регрессионная задача. Необходимо реализовать любую **глубокую NLP-модель**. Допускается использование как **RNN**, так и **моделей на основе трансформеров**.

## Дополнительные (необязательные) задачи

Хороший балл можно получить, выполнив только основную часть. Но есть возможность заработать дополнительные баллы:

### 1. Тематическое моделирование текстов

- Проведите **кластеризацию** отзывов из колонки `review`.
- Получите **текстовые эмбеддинги** с помощью англоязычных моделей (поскольку тексты на английском).
- Кластеризацию можно выполнить, например, с помощью **C-TF-IDF**, как показано [в этом подходе](https://maartengr.github.io/BERTopic/api/ctfidf.html).
- Оценка качества кластеров производится **эмпирически**: хорошие кластеры содержат слова, консистентно описывающие одну тему.
- Альтернативный способ: использовать **LDA** на эмбеддингах от трансформеров.

### 2. Дистилляция модели

- Попробуйте **дистиллировать трансформер в LSTM**.
- В качестве отправной точки можно взять [этот код](https://github.com/huggingface/transformers/blob/main/examples/research_projects/distillation/distill.py) или воспользоваться готовым решением.
